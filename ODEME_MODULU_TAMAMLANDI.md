## Önemli Not: Next.js Sunucusunu Yeniden Başlatın

Eğer "prisma://" hatası alıyorsanız:

1. **Next.js sunucusunu durdurun** (Ctrl+C)
2. **.next klasörünü temizleyin** (zaten temizlendi)
3. **Sunucuyu yeniden başlatın:**
   ```powershell
   npm run dev
   ```

## Kullanım

1. Sol menüden "Ödeme" > "Ödeme Ekle" seçin
2. Formu doldurun ve kaydedin
3. Ödemeleri listeleyin, düzenleyin veya silin
